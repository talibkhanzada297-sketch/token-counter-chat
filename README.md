# 🤖 Token Counter Chat

Real-time token counter for AI chat applications. Monitor API usage and costs with support for GPT, Claude, and other models.

## Features

- Real-time token counting
- Multiple model support (GPT-3.5, GPT-4, Claude)
- Cost estimation
- Message history
- Export to JSON
- Responsive design
